<h2 id="overview">Overview</h2>
<p>This is the official website landing page for <strong>Perteawi (Perkebunan Bumi Pertiwi)</strong>, an Indonesian premium tea producer. The page is built as a Vue.js 2 application with TypeScript, using Bootstrap-Vue and following atomic design patterns.</p>

<h3 id="1-core-values">1. Core Values</h3>
<p>The brand is built on these three pillars:
✅ <strong>Premium Quality Tea</strong> - Uses only selected high-grade tea leaves
✅ <strong>Traditional Plantation Revitalization</strong> - Sustains heritage farming practices
✅ <strong>Farmer Welfare</strong> - Ensures positive social impact for traditional tea farmers</p>
<h3 id="2-product-standard">2. Product Standard</h3>
<p>Perteawi uses <strong>Golden Flush / Peko grade tea</strong>:</p>
<ul>
<li>Only harvests closed tea buds (peko) plus the top 2 adjacent leaves</li>
<li>Uses Camelia sinensis cultivar</li>
<li>This is the industry definition for premium grade tea</li>
</ul>
<h3 id="3-design-technical-implementation">3. Design &amp; Technical Implementation</h3>
<ul>
<li>Fully responsive design optimized for mobile, desktop and large 4k+ displays</li>
<li>Responsive image loading (WEBP + PNG fallbacks with 1x / 2x pixel density)</li>
<li>Modular atomic component architecture (Atoms / Molecules / Organisms / Sections)</li>
<li>Bilingual support (Indonesian + English) with inline i18n</li>
<li>SCSS styling with design system variables and mixins</li>
<li>Bootstrap grid layout system</li>
</ul>

<h2 id="business-positioning">Business Positioning</h2>
<p>This landing page positions Perteawi as an ethical premium Indonesian tea brand that combines high product quality with social responsibility towards traditional farming communities. The visual design uses natural earth-tone branding, authentic plantation photography and clean minimalist layout.</p>
