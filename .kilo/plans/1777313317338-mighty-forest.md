# Plan: Enhanced `.group-title` styling in portfolios page

## File
`src/pages/portfolios/index.astro`

## Current State (lines 121-140)
```scss
.group-title {
    font-size: 1.25rem;
    font-weight: 600;
    letter-spacing: 0.05em;
    color: var(--color-text-secondary);
    text-transform: uppercase;
    position: relative;
    padding-bottom: 0.75rem;

    &::after {
      content: '';
      position: absolute;
      bottom: 0;
      left: 50%;
      transform: translateX(-50%);
      width: 2rem;
      height: 2px;
      background: var(--color-accent);
    }
  }
```

## Issues
- `color: var(--color-text-secondary)` — visually recessive, doesn't command attention as a section header
- Centered underline feels dated and doesn't align with the Double-Bezel / pill aesthetic used elsewhere
- No Double-Bezel outer shell wrapping the group header, inconsistent with the rest of the design system
- `letter-spacing: 0.05em` with `text-transform: uppercase` is acceptable but lacks refinement
- No hover or interactive feedback (though it's a static heading, so this is acceptable)

## Proposed Enhancements

### Option A: Pill-shaped group badge (consistent with EyebrowTag)
Wrap the group title in a pill-style container similar to `EyebrowTag` but larger — uses a glass-morphism pill with subtle border and accent underline.
- Outer: `display: inline-flex`, `border-radius: var(--radius-full)`, `padding: 0.5rem 1.25rem`, hairline border
- Inner: `background: var(--color-glow-1)` with `color: var(--color-accent)` or a stronger `var(--color-text-primary)`
- Replace the centered `::after` underline with an inset bottom accent bar inside the pill

**Pros**: Matches EyebrowTag pattern. Premium pill aesthetic. Consistent with nav pill.
**Cons**: If there are many group titles, pills might feel visually heavy.

### Option B: Editorial left-bar accent
Remove the centered underline. Add a left-side accent bar (2px thick, accent color) that sits to the left of the text, with generous left padding (`padding-left: 1rem`).
- Align left within the centered container
- `border-left: 2px solid var(--color-accent)` with `padding-left: 0.75rem`
- Larger font: `clamp(1.25rem, 2vw, 1.5rem)`, `font-weight: 700`
- `color: var(--color-text-primary)` for more authority

**Pros**: Editorial, sophisticated. Clean visual hierarchy.
**Cons**: Breaks center alignment of the group header.

### Option C: Double-Bezel minimal badge (Recommended)
Use the Double-Bezel outer shell + inner core architecture around the group title, making it a self-contained premium badge.
- Outer: `display: inline-block`, `border-radius: var(--radius-lg)`, `padding: 0.25rem`, hairline border
- Inner: `border-radius: calc(var(--radius-lg) - 0.25rem)`, `padding: 0.5rem 1.25rem`, `background: var(--color-surface-card)`
- Title text: `font-size: 0.8125rem`, `font-weight: 600`, `letter-spacing: 0.12em`, `text-transform: uppercase`, `color: var(--color-accent)`
- Removes the `::after` underline — the bezel shell itself provides the visual boundary
- Add subtle ambient shadow that lifts on hover

**Pros**: Directly matches the Double-Bezel pattern used everywhere. Lightweight. Consistent.
**Cons**: Slightly more markup (need a wrapper div).

## Decision: Option C — Double-Bezel minimal badge

### Changes to `portfolio-group-header` and `.group-title`

**HTML change** — wrap the h2 in a Double-Bezel structure:
```astro
<AnimatedEntrance class='portfolio-group-header' delay={String(gi * 200)}>
  <div class="group-bezel-outer">
    <div class="group-bezel-inner">
      <h2 class='group-title'>{group.label}</h2>
    </div>
  </div>

```

**CSS changes** — replace current `.group-title` and `.portfolio-group-header`:
